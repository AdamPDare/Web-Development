Creating reactive Columns

To create three reactive equal columns, on this webpage, I set the width of each column to 33.33% and to cover 100% of the width. Furthermore, I used the @media screen feature to set the maximum width of the screen and so that the column positioning is affected by the size of the window making the webpage more compatible with mobile devices. Therefore, when the window is a specific size the columns and the navigation bar becomes vertical.Below it shows the CSS code to create the reactive columns

<h3>Code Extract</h3>
<style>
    code{
            
        .columnn{
            float:left;
            width:33.33%;
            padding: 15px;
        }
                
        .row::after{
            content:"";
            display:table;
            clear:both;
        }

        @mediascreen and (max-width: 600px){
            .column{
            width: 100%;
        }
    }
<style>

Navigation Bar with Dropdown

To create a navigation bar, I needed to used an unordered and normal list, the hover function and a dropdown button. These features allowed me to create a navigation bar that allows you to navigate each webpage on my website so far. I also utilised the @mediascreen feature to allow the navigation bars size change according to the size of the screen enabling my webpage to be used on a mobile device.

<h3>Code Extract</h3>
<style>
    ul{
        list-style-type:none;
        margin: 0;
        padding: 0;
        overflow: hidden;
        background-color: var(--main-bg-color);
    }

    li{
        float: left;      
        border-right: 1px solid var(--txt-border-color);
    }

    li a, .dropbtn{
        display: inline-block;
        color: var(--txt-border-color);
        text-align: center;
        padding: 14px 16px;
        text-decoration: none;
        
    }

    li a:hover:not(.active){
        border-bottom: 3px solid var(--txt-border-color);
        border-top: 2px solid var(--txt-border-color);

    }
    li a:hover, .dropdown:hover .dropbtn{
        border-bottom: 3px solid var(--txt-border-color);
    }

    li.dropdown{
        display: inline-block;
    }

    .dropdown-content{
        display: none;
        position: absolute;
        background-color: var(--main-bg-color);
        min-width: 160px;
        box-shadow: 0px 8px 16px 0px rgba(0,0,0,0.2);
        z-index: 1;
    }

    .dropdown-content a{
        color:var(--txt-border-color);
        padding: 12px 16px;
        text-decoration: none;
        display: block;
        text-align: left;
    }

    .dropdown-content a:hover{
        border-bottom: var(--txt-border-color);
    }
    .dropdown:hover .dropdown-content{
        display: block;
    }
    @media screen and (max-width: 600px) {
        .Navigation a{
            float:none;
            display: block;
            width: 100%;
            text-align: left;
        }
    }
