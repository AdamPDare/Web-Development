Creating reactive Columns

To create three reactive equal columns, on this webpage, I set the width of each column to 33.33% and to cover 100% of the width. Furthermore, I used the @media screen feature to set the maximum width of the screen and so that the column positioning is affected by the size of the window making the webpage more compatible with mobile devices. Therefore, when the window is a specific size the columns and the navigation bar becomes vertical.

<h3>Code Extract</h3>
<style>
    code{
            
        .columnn{
            float:left;
            width:33.33%;
            padding: 15px;
        }
                
        .row::after{
            content:"";
            display:table;
            clear:both;
        }

        @mediascreen and (max-width: 600px){
            .column{
            width: 100%;
        }
    }
<style>
